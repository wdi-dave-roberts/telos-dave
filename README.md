## The System

This isn't a document you fill out once. It's a mirror that gets clearer over time.

**Weekly:** Reid sends targeted journal questions (Sunday 7pm)
**Daily:** New captures get processed through Fabric analysis
**Bi-weekly:** Full telos runs through blindspot + contradiction analysis (Friday 6pm)
**Monthly:** Everything synthesized into a narrative (1st of month)

See `GUIDE.md` for the full system overview.
